# MedLog User Guide

## Getting Started

### Account Setup
1. Register with institutional email
2. Verify email via link
3. Complete profile (specialty, year)
4. Join institution code (if applicable)

### Dashboard Overview
- **Cases:** View/create patient cases
- **Analytics:** Track learning progress
- **References:** Quick lookup tools
- **Settings:** Account preferences

## Core Features

### Creating Cases
1. Click "New Case"
2. Enter patient data (de-identified)
3. Add images, notes, codes
4. Save or share

### Case Templates
- Pre-built templates by specialty
- Customize your own
- Share with institution

### AI Analysis
- Automatic gap identification
- Learning recommendations
- Progress tracking

## Collaboration

### Sharing Cases
- Generate share links
- Time-limited access
- Password protection option

### Institution Features
- Resident tracking
- Custom templates
- Analytics dashboard

## Mobile Access
- Responsive design
- Offline case viewing
- Push notifications
